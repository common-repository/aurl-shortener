== Description ==
AURL Shortener plugin helps you to shorten the url of your posts or pages as fast as possible! 
AURL Shortener is fast, quick, includes support, and most importatnly it is free!
== Installation ==

The installation is very easy!

1. Upload the plugin in wordpress
2. Activate the plugin
3. Go to settings and add your API Key from AURL.XYZ
4. Use shortcode [shorten] on any page or posts you want to shorten the url. It generates the URL and shows the url where ever you put the shortcode.
